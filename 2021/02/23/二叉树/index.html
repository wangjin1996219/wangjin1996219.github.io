二叉树的遍历（三种）

根据根节点的访问顺序可以将二叉树的遍历分为：

前序遍历：根节点→左子树→右子树

中序遍历：左子树→根节点→右子树

后序遍历：左子树→右子树→根节点

注意：

  • 以根访问顺序决定是什么遍历
  • 左子树一般都是先于右子树的

二叉树的每种遍历方式都可以分为递归和非递归两种，其中非递归的方式一定要掌握！

前序遍历

前序递归：

前序非递归：（非递归都是用辅助栈实现）

  • 节点 root 先到达最左端的叶子节点并将路径上的节点添加到res集合中、入栈；

  • 然后每次从栈中弹出一个元素后，root 到达它的右孩子，并将右孩子看作 root上面的步骤。

因为在到达左边叶子节点的过程中是先添加、后入栈，所以右节点作为 root 节点打印时，它的根节点以及根节点的左节点是已经打印的，满足根→左→右的顺序

中序遍历

中序递归：

中序非递归：（非递归都是用辅助栈实现）

  • 节点 root 先到达最左端的叶子节点并将路径上的节点入栈；
  • 然后每次从栈中弹出一个元素后添加到res集合中，root 到达它的右孩子，并将右孩子看作 root上面的步骤。

因为在到达左边叶子节点的过程中是先添加、后入栈，所以右节点作为 root 节点打印时，它的根节点以及根节点的左节点是已经打印的，满足左→根→右的顺序

后序遍历

后序遍历：

后序非递归：（非递归都是用辅助栈实现）

  • 节点 root 先到达最右端的叶子节点并将路径上的节点入栈；
  • 然后每次从栈中弹出一个元素后添加到res集合中，root 到达它的左孩子，并将左孩子看作 root上面的步骤。

因为在到达右边叶子节点的过程中是先添加、后入栈，所以左节点作为 root 节点打印时，它的根节点以及根节点的右节点是已经打印的，满足根→右→左的顺序

后序遍历的顺序为：左→右→根，所以将res集合反转就是后序遍历的结果

题目二

热题100-124. 二叉树中的最大路径和

路径 被定义为一条从树中任意节点出发，沿父节点-子节点连接，达到任意节点的序列。同一个节点在一条路径序列中 至多出现一次 。该路径 至少包含一个 节点，且不一定经过根节点。

路径和 是路径中各节点值的总和。

给你一个二叉树的根节点 root ，返回其 最大路径和 。

解题思路

实现一个函数 maxPathSumImpl（TreeNode root）：该函数计算二叉树中的一个节点的最大贡献值，具体而言，就是在以该节点为根节点的子树中寻找以该节点为起点的一条路径，使得该路径上的节点值之和最大。

实现思路就是：

递归左子树，如果左子树的返回值大于零返回该结果，反之返回零（负数还不如不选择）

递归右子树，如果右子树的返回值大于零返回该结果，反之返回零

（也就是说，当左子树的返回值为零，就不走左子树，右子树同理，当左子树和右子树的返回值同时为零，两边都不走）

返回该节点和其左子树、右子树中较大值的和

因为最大路径和有可能出现在树中局部的一棵子树上，所以要对中间结果进行记录：

定义一个全局变量 tempMaxSum

在每个节点的递归中，计算以该节点为根节点的子树最大路径和：

一个子树中的最大路径和 = 左子树提供的最大路径和 + 根节点值 + 右子树提供的最大路径和

题目三

热题100-剑指 Offer 68 - II. 二叉树的最近公共祖先

给定一个二叉树, 找到该树中两个指定节点的最近公共祖先。

解题思路

二叉树的最近公共祖先：有一个节点的左右子树中同时包含要查找的两个节点

递归实现流程：

  • 当超出叶子结点、或者该节点正好是要查找的某一个节点时，返回节点本身

  • left=左子树递归

  • right=右子树递归

  • 递归结果的可能组合：

    同时不为null，也就是该节点就是最近的公共节点，返回节点本身（一定要最先判断，否则就会返回单边不为null的情况）

    有一个不为null，返回不为null的那个

    剩下的情况就是都为null，返回null

题目四

牛客-输出二叉树的右视图

请根据二叉树的前序遍历，中序遍历恢复二叉树，并打印出二叉树的右视图

解题思路

重建二叉树的思路：

  • 递推参数：根节点在前序遍历的索引root、子树在中序遍历的左边界left、子树在中序遍历的右边界right；

  • 终止条件：当left>right，代表已经越过叶节点，此时返回 null；

  • 递推工作：

    建立根节点 node ： 节点值为 preorder[root] ；

    划分左右子树： 查找根节点在中序遍历 inorder 中的索引 i ；（主要根据中序遍历找到左子树的节点数，进而在前序中找到右子树的根节点，这里中序遍历中的右下标主要用于判断是否越界）

输出右视图的思路：

  • 二叉树的右视图就是从右边看过去能看到的第一个节点
  • 所以根据层序遍历的思路，每次输出每层的最后一个节点即可

题目八

牛客-判断t1树中是否有与t2树拓扑结构完全相同的子树

给定彼此独立的两棵二叉树，判断 t1 树是否有与 t2 树拓扑结构完全相同的子树。

设 t1 树的边集为 E1，t2 树的边集为 E2，若 E2 等于 E1 ，则表示 t1 树和t2 树的拓扑结构完全相同。

解题思路

若树 root2 是树 root1 的子结构，则子结构的根节点可能为树 root1 的任意一个节点。

方法isContains：遍历树 root1 中的每个节点NodeA

  • 特例处理： 当 树 root1 为空 或 树 root2 为空 时，直接返回 false

  • 返回值： 若树 root2 是树 root1的子结构，则必满足以下三种情况之一

    调用isContainsImpl方法，以 节点 root1 为根节点的子树包含树 root2 ，返回true

    左子树中发现子结构root2

    右子树中发现子结构root2（所以用||连接）

方法isContainsImpl：判断树 root1 中 以 NodeA 为根节点的子树 是否包含树 root2

  • 终止条件：

  • 当节点root1与root2同时为空时，说明树 root2 已匹配完成（越过叶子节点），因此返回 true

    当节点 root1 为空或者root2为空时，两者不等，返回 false

    当节点 root1 和 root2 的值不同：说明匹配失败，返回 false ；

  • 返回值：

    判断 root1 和 root2 的左子节点是否相等，即isContainsImpl(root1.left, root2.left) ；

    判断 root1 和 root2 的右子节点是否相等，即 isContainsImpl(root1.right, root2.right) ；
