基本的查询语句

  1. 查询一列：SELECT cust_id FROM customers;

  2. 查询全部列：SELECT cust_id FROM customers；

  3. LIMIT 第一个数为开始的行数，第二个数为要检索的行数

    • 从第0行（代表第一行）开始：SELECT cust_name FROM customers LIMIT 5;
    • 从第5行（实际表中的第六行）开始，显示5行：SELECT cust_name FROM customers LIMIT 5, 5;

  4. 排序指令

    • 默认ASC降序：SELECT prod_name FROM products ORDER BY prod_price;
    • DESC升序：SELECT prod_name FROM products ORDER BY prod_price DESC;
    • 按多个列排序 先按价格，再按名称：SELECT prod_name FROM products ORDER BY prod_price, prod_name;
    • DESC 降序,然后再取第0行：SELECT prod_name FROM products ORDER BY prod_price DESC LIMIT 1;

  5. WHERE 的逻辑运算符有AND ,OR, NOT,IN

    • 检索出由1003或1002供应商生成的商品：SELECT * FROM products WHERE vend_id = 1003 AND prod_price <= 10;

  6. MySQL提供了5种的聚集函数：

    • AVG( ) 返回某列的平均值：SELECT AVG(prod_price) FROM products;
    • COUNT( ) 返回某列的行数
    • MAX( ) 返回某列的最大值
    • MIN( ) 返回某列的最小值
    • SUM( ) 返回某列值之和

  7. 内链接inner join on：select * from a_table a inner join b_table bon a.a_id = b.b_id;

  8. 左连接left join on / left outer join on：select * from a_table a left join b_table bon a.a_id = b.b_id;

  9. 右连接（右外连接）right join on / right outer join on：select * from a_table a right outer join b_table b on a.a_id = b.b_id;
